# 一、系统说明

基于springboot+vue+elementui+echarts开发的产业园区智慧公寓管理系统,系统功能齐全, 代码简洁易懂，适合小白学编程。

# 二、系统架构

######      前端：vue| elementui |echarts

######      后端：springboot | mybatis 

######      环境：jdk1.8+ | mysql8.0+ | maven
